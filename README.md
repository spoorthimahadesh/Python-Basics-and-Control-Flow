# Python-Basics-and-Control-Flow

📊 Student Grade Tracker (Jupyter Notebook Project)
📌 Overview

This project is a Student Grade Tracker and Analysis System built using Python in a Jupyter Notebook. It processes raw student data, cleans it, validates inputs, and performs basic analysis such as grading, averages, and predictions.

🚀 Features
✅ Data cleaning (names, roll numbers, marks)
✅ Validation of student names
✅ Conversion of marks from string to list
✅ Grade calculation (A+, A, B, C, F)
✅ Report card generation
✅ Average calculation
✅ Pass/Fail prediction
✅ Basic machine learning model (classification)
🛠️ Technologies Used
Python 🐍
Jupyter Notebook 📓
pandas
scikit-learn
📂 Dataset

The dataset contains:

Student names
Subject-wise marks
Attendance percentage
Study hours
Pass/Fail labels
